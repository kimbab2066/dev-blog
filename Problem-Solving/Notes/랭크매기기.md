### **2ï¸âƒ£ Ordinal Ranking (ì—°ì† ë­í‚¹)**

```python
def ordinal_ranking(emergency):
    er = sorted(emergency, reverse=True)
    rank = {}
    prev_score, prev_rank = None, 0
    for i, v in enumerate(er):
        if v != prev_score:
            prev_rank = i + 1
        rank[v] = prev_rank
        prev_score = v
    return [rank[x] for x in emergency]
```
